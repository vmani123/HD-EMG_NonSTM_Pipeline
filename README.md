Contains 2 main codefiles:
  1. The Python code for setting up a basic TCP server, receiving incoming data, and plotting
     the incoming data on a graph. Utilizes sockets interfacen.
  2. The ESP32 C code for setting up a basic TCP client and a SPI master. The SPI master controller
     receives incoming data and sends that data over TCP to a server. This code
     was developed using the esp-idf extension in Visual Studio Code.
